# 🎮 Office Chaos

A fast-paced 2D arcade game built with Python and Pygame.

You control a rolling office chair trying to survive the chaos:
- ☕ Collect coffee for energy
- 🐤 Grab rubber ducks for bonus points
- 📧 Avoid emails (they drain your score!)
- 😡 Don’t get caught by the manager!

---

## 🚀 Features

- Smooth physics-based movement
- Dash mechanic
- AI manager that chases the player
- Procedural gradient + grid background
- Sound effects
- Score system

---

## 🛠 Tech Stack

- Python 3
- Pygame

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/eya-lachhab/office-chaos.git
cd office-chaos
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the game:

```bash
python main.py
```

---

## 🎯 Controls

- Arrow Keys → Move
- Space → Dash
